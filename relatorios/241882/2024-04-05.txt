Tarefa:  
"obj_video_IMP.obtem_dimensoes_do_arquivo": Usar "ffmpeg" ou similar para obter os
parâmetros do vídeo a partir do arquvo em disco.


Relatório:

Para implementação do que está definido na tarefa foi necessário, primeiramente, efetuar o download do
repositório, criar um virtual environment utilizando a biblioteca venv, do Python, onde neste último passo
tornamos possível a instalação das dependências necessárias para executar o projeto.

Ao tentar executar o servidor, foi notado a existência de erros devido a alguns imports inexistentes. Para corrigir
tal problema, foram executadas as seguintes correções:

1- Trocar os imports com nome prefixado com comando_ver_videos_{...} por comando_ver_videos_de_usuario_{...}
2- Importar obj_sessao e obj_usuario no arquivo comando_solicitar_pag_buscar_videos_IMP.py

Após isso se tornou possível executar o servidor.

Partindo então para a implementação da tarefa, de fato, foi adicionada a função obtem_dimensoes_do_arquivo no arquivo
obj_video_IMP.py e sua devida interface no arquivo obj_video.py.

Na função obtem_dimensoes_do_arquivo checamos se um video com um dado nome de arquivo {arq}.mp4 existe em disco, e
retornamos None caso não exista. Caso o arquivo exista, utilizamos o módulo subprocess do python para executar o comando
ffprobe, para obter as informações desejadas sobre o arquivo de vídeo. 

Dada uma execução com sucesso, a função obtem_dimensoes_do_arquivo retorna uma tupla do tipo (width, height), que representa as
dimensões do arquivo de vídeo.
